São ações executadas assim que são chamadas ou em decorrência de algum evento.
Pode receber parâmetros e retornar um resultado.
Em funções há a Chamada, o Parâmetro e o Retorno.
Sintaxe:
function name(parameter1, parameter2, parameter3, ...) {
  // code to be executed
}

Exemplos:
'function soma(n1=0, n2=0) {
    return n1 + n2
}
console.log(soma(3, 4))'

soma(3, 4) - é a CHAMADA;
3, 4 - são os parâmetros;
n1 + n2 - é o que vai retornar após eu fizer a chamada e passar os parâmetros.


METODO
Quando as funções recebem as propriedades de um objeto.
  .push() - Adiciona elementos a uma array.
  .reverse() - Inverte a ordem dos elementos..
